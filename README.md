## 최종 선정 모델 : Claude Sonnet 4.6
## 선정 이유 : 
간단한 회의록 정리 및 보고서 초안은 Gemini가 빠르나 ‘경쟁력 있는 문서’로 만드는 것은 Glaude가 낫다
보고서 초안 요청 시 세 모델모두 속도는 비슷하였으나 Claude의 경우 적당히 아이콘을 사용하여 시각적으로 보기도 좋고 간결하게 요약하여 가독성이 좋았음. 
그리고 첨부된 여러개의 파일을 통해 내용을 요약하고 보고서 형태의 최종 결과물에서도 한국어의 자연스러운 문체와 더불어 표와 그래프의 표현, 한자 및 아이콘 사용등이 적절하게 이용되어 전체적인 다른 모델에 비해 가독성이 높음.

데이터는 정확한 출처 및 근거를 제시한다.
정확하지 않는 내용은 모른다고 말한다

### 시스템 프롬프트
#### v1. 
첨부한 파일의 내용을 포함한 네이버 실적을 매출, 영업이익, 가이던스 중심으로 요약해 줘

#### v2.
첨부한 파일의 내용을 포함한 네이버의 실적을 매출, 영업이익, 가이던스 중심으로 임원진에게 보고할 보고서 형태로 A4 3페이지 분량으로 작성해 줘.

#### v3.
당신은 증권사 전략 기획팀 소속의 시니어 애널리스트입니다.
아래의 조건에 맞게 첨부한 파일의 내용을 포함한 네이버의 매출, 영업이익, 가이던스를 중심으로 내용을 요약하여 보고서 형식으로 작성해 줘.
-	목적 : 네이버의 2026년 2분기 가이던스 파악
-	대상 : 임원진(비 기술직)
-	분량 : A4 3페이지 분량
-	필수항목 : 현 상황과 실적, 출처를 명시할 수 있는 정보만 제공
-	어조(톤) : 간결하고 객관적인 비즈니스 문체
-	출력 형식 : 두괄식
